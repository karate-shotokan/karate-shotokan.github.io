Cria o design UX/UI completo de um website moderno, elegante e altamente credível para uma escola/estrutura de Karaté Shotokan em Portugal, liderada pelo Sensei Carlos Cardoso, com ligação à ANAM (Associação Nacional de Artes Marciais).

OBJETIVO DO SITE
O site deve servir simultaneamente para:
1. transmitir autoridade, tradição, disciplina e credibilidade;
2. atrair novos alunos (crianças, jovens e adultos);
3. apresentar o Sensei, a filosofia e a prática do Karaté Shotokan;
4. organizar informação de vários dojos/localizações;
5. divulgar eventos, exames, estágios, competições e notícias;
6. facilitar contactos e inscrições;
7. permitir futura manutenção simples por conteúdo estruturado.

CONTEXTO DE MARCA
O Karaté Shotokan deve ser representado com equilíbrio entre tradição e modernidade.
O visual não deve parecer nem antiquado nem demasiado comercial/fitness.
Deve transmitir:
- disciplina
- respeito
- foco
- força controlada
- clareza
- confiança
- pedagogia
- comunidade

ESTILO VISUAL
Criar um visual premium, limpo e institucional, mas acolhedor.
Direção estética:
- minimalista
- elegante
- forte uso de espaço em branco
- grelha organizada
- navegação simples
- tipografia sólida e legível
- fotografia real e autêntica de treino, dojos, graduações, eventos e praticantes
- evitar excesso de ornamento japonizante cliché

PALETA SUGERIDA
- base clara: branco / off-white / cinza muito claro
- contraste: preto / carvão
- cor de destaque principal: vermelho profundo / bordô inspirado em tradição marcial
- acentos opcionais discretos: dourado muito subtil para distinção, graduações ou detalhes institucionais
- o design deve funcionar bem em desktop e mobile

CONTEÚDO ESTRUTURÁVEL
O design deve assumir que o site será implementado em HTML, Bootstrap, JavaScript, jQuery e GitHub Pages.
Por isso, criar a UX e a arquitetura visual de forma que o conteúdo possa ser alimentado por ficheiros JSON.
Assumir que:
- os textos poderão vir de ficheiros JSON
- os dojos serão alimentados por JSON
- eventos/notícias serão alimentados por JSON
- galerias serão alimentadas por JSON + imagens em pastas
- os componentes devem funcionar bem com dados dinâmicos simples

IMPORTANTE PARA IMPLEMENTAÇÃO
Criar um design que seja:
- simples de converter para HTML + Bootstrap
- simples de manter
- modular
- altamente reutilizável
- ideal para GitHub Pages

INSPIRAÇÃO FUNCIONAL
Inspirar-se nas melhores práticas de websites de artes marciais:
- imagem forte
- organização simples
- páginas para graduações/syllabus
- etiqueta e instruções para iniciantes
- conteúdos multimédia úteis
- múltiplos locais/dojos
- provas sociais, eventos e resultados
- equilíbrio entre tradição e conversão
